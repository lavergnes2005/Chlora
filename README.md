# Chlora

*A science-fiction JRPG about ecology, curiosity, and the meeting of two
worlds.*

------------------------------------------------------------------------

## Overview

Chlora is an original game project centered on exploration, scientific
discovery, and symbiosis rather than conquest. The project is documented
as a living design library so that worldbuilding, gameplay, narrative,
and art direction remain consistent throughout development.

------------------------------------------------------------------------

## Getting Started

If you're new to the project, read the documentation in this order:

1.  **00_Project_Constitution.md** --- The project's guiding philosophy.
2.  **01_Canon_Bible.md** --- The authoritative source of established
    facts.
3.  **02--09** --- Reference documents (art, world, biology, characters,
    culture, gameplay, story).
4.  **10--14** --- Support documents (glossary, design journal,
    development guide, research, roadmap).

## Documentation Hierarchy

When documents disagree, precedence is:

1.  Project Constitution
2.  Canon Bible
3.  Reference Documents
4.  Support Documents

Conversation history is **never** considered authoritative.

------------------------------------------------------------------------

## Core Principle

> **One concept. One authoritative location.**

Each concept should have a single document that owns it. Other documents
should reference that source rather than duplicate information.

------------------------------------------------------------------------

## Contributing

Before adding new content:

1.  Check the Project Constitution.
2.  Verify whether the idea is already canon.
3.  Update the owning document.
4.  Record significant design rationale in the Design Journal.

Markdown files are the project's source of truth, and Git provides the
revision history.
