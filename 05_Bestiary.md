# CHLORA PROJECT

# 05 --- Bestiary

**Version:** 1.0\
**Status:** Living Reference\
**Tier:** Biology Reference

------------------------------------------------------------------------

## Purpose

The Bestiary catalogs the organisms of the Wilds and their relationship
to the ecosystem.

Unlike the Biology Bible, which defines the rules of life, the Bestiary
documents the individual species and lineages that emerge from those
rules.

This document will grow continuously throughout development.

------------------------------------------------------------------------

## Scope

### This document defines

-   Organism profiles
-   Evolutionary inspirations
-   Ecological niches
-   Behavior
-   Human interactions

### This document does **not** define

-   Biological laws (Biology Bible)
-   World geography (World Bible)
-   Gameplay mechanics (Game Design Document)
-   Story events (Story Bible)

------------------------------------------------------------------------

# Design Philosophy

Every organism should originate from one or more real biological
inspirations before evolving into a distinct Wild lineage.

Players should recognize the underlying biological principle---not the
original Earth species.

# Ecological Roles

Every future organism should fulfill one or more ecological roles:

-   Primary producer
-   Grazer
-   Filter feeder
-   Predator
-   Parasite
-   Mutualist
-   Pollinator
-   Scavenger
-   Decomposer
-   Ecosystem engineer
-   Keystone species
